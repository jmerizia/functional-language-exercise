WASM Functional Language

This is an implementation of a lisp-like functional language with lazy evaluation and first-class functions. I ported it to WASM with Emscripten, so it can be used completely within the browser! See the grammar for help.

Source code: https://github.com/jmerizia/functional-language-exercise

Limitations

Some of the error messages from the tokenizer aren't very helpful. It would be nice to have error messages with line numbers.

The language is very simple, and I haven't had the time to build up the standard library. Perhaps I'll work on that some other time.
